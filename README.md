# Maze-solver
Maze Solver Using A*search Algorithm for AI Internship Project. Finds the shortest path from start to goal in a maze while avoiding obstacles using intelligent heuristic search.

## Features
-Shortest path finding
-Avoids Obstacles
-Visual path output
-Python implementation

## Technologies Used
-Python
-A* Algorithm
-Heap Queue

## Applications
-Robotics Navigation
-Game AI movement
-GPS route Finding
-Maze solving system
